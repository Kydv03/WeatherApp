## Weather-App
A simple **Weather Dashboard** web application built using **HTML, CSS, and JavaScript**.  
It fetches real-time weather data using the **OpenWeatherMap API** and displays it in a user-friendly interface.

## Features

- Search weather by **city, address, or zip code**  
- Current temperature, humidity, and wind speed  
- Displays **weather condition icons** (Clouds, Clear, Rain, etc.)  
- Responsive layout for **desktop and mobile**  
- Automatic location detection using **Geolocation API**

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)  
- OpenWeatherMap API
